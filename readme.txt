CalicoTrivia

Calico Trivia es una pagina dedicada como su nombre lo dice a un juego de trivia, en donde el usuario se puede registrar o iniciar sesión y comenzar a jugar respondiendo preguntas de las cuales disponen de 4 posibles respuestas donde 1 es la correcta. La puntuación de cada usuario se muestra en el apartado de "clasificación"

Características

-Conexión a Api con preguntas
-Panel para crear/iniciar sesión con cuentas
-Interfaz de clasificaciones
-responder preguntas
-login personalizado para admin > redireccion a django admin

Tecnologías Utilizadas

Django, Python, Api propia HTML/CSS

Instalación:
instalar python si no lo tiene desde la pagina oficial
https://www.python.org/

Clonar el repositorio o descargar directamente de : git clone https://github.com/d4glock/trivia.git

Navegar al directorio del proyecto: cd trivia_project Instalar las dependencias:

pip install djangorestframework
Ejecutar el servidor:

python manage.py runserver
